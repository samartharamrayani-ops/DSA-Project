# DSA-Project
#  Academic Planner – DSA Java Project

A console-based Academic Planner application built using Java and Data Structures.



#  Project Overview

The **Academic Planner** helps students organize academic tasks such as assignments, deadlines, and schedules.
It applies DSA concepts to efficiently manage and process academic data.

The application allows users to:

* Add assignments or tasks
* View stored assignments
* Organize tasks efficiently
* Automatically generate timelines
* Manage data using DSA structures


#  Technologies Used

* **Language:** Java
* **Concepts:** Data Structures & Algorithms
* **IDE:** VS Code / IntelliJ / Eclipse
* **Type:** Console Application


#  Features

 Add new academic tasks
 View stored tasks
 Organize tasks efficiently
 Automatic timeline generation
 Demonstrates multiple DSA concepts
 Simple console-based interface

#  How to Run

### 1️ Clone the Repository

```
git clone https://github.com/your-username/AcademicPlannerDSA.git
```

### 2️ Navigate to the Project Folder

```
cd AcademicPlannerDSA
```

### 3️ Compile the Program

```
javac AcademicPlannerDSA.java
```

### 4️ Run the Program

```
java AcademicPlannerDSA
```
